# 🐾 PawPrint — 快速上手

> 不需要写代码。让你的 AI Agent 帮你搞定一切。

---

## PawPrint 是什么？

PawPrint 是一个**私密文档分享工具**。你写好文档，设个密码，发个链接给别人 —— 就这么简单。

跟 Notion、Google Docs 最大的区别：**你的 AI Agent 可以直接帮你发布文档**。不用复制粘贴，不用手动操作。

---

## 3 步开始

### 第 1 步：部署你自己的 PawPrint

告诉你的 OpenClaw Agent：

> "帮我部署一个 PawPrint，用我的 GitHub 账号和 Vercel。"

Agent 会自动帮你：
- ✅ 创建 GitHub 仓库
- ✅ 部署到 Vercel
- ✅ 配置好所有东西
- ✅ 给你一个可访问的链接

### 第 2 步：发布文档

写好一篇文档（或者让 Agent 写），然后告诉它：

> "把这篇调研报告发布到 PawPrint，分类放到项目里，设个密码 abc123。"

Agent 会自动：
- ✅ 把文档放到正确的位置
- ✅ 设置密码
- ✅ 推送并部署
- ✅ 给你分享链接

### 第 3 步：分享

把链接发给任何人。他们打开链接，输入密码，就能看到文档。

## 你能做什么？

### 📝 分享文档
> "发布这篇会议纪要到 PawPrint。"

### 🔒 设置密码
> "给这篇文档设个密码 mysecret。"

### 🛡️ 加密敏感内容
> "用端到端加密发布这篇文档，密钥是 topkey。"

分享链接会自带密钥，收到链接的人直接打开就能看。你不给链接，谁都看不到。

### 🔥 阅后即焚
> "发布这篇文档，24 小时后自动过期。"

### 📧 收集邮箱
> "发布这篇文档，要求读者先输入邮箱才能看。"

### 📊 查看谁看了
> "查看 PawPrint 的阅读数据。"

Agent 会告诉你：谁看了、看了多久、看到哪里。

你也可以自己打开 `/admin` 看仪表盘。

## 常见问题

**Q: 我需要会写代码吗？**
A: 不需要。所有操作都可以通过跟 Agent 对话完成。

**Q: 免费吗？**
A: 完全免费。部署在 Vercel 免费套餐上，没有任何费用。

**Q: 我的文档安全吗？**
A: 你的文档存在你自己的 GitHub 私有仓库里。密码在服务端验证，不会暴露在浏览器。E2E 加密的文档连服务器都看不到内容。

**Q: 能用手机看吗？**
A: 可以，文档页面自适应移动端。

**Q: 我能自定义域名吗？**
A: 可以，在 Vercel 设置里绑定你自己的域名就行。
