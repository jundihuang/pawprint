## 一、产品概述

一套**以 Shopify 为核心**的 AI 工具集，作为 Shopify App 安装到商家后台，覆盖独立站卖家从选品到投放、从内容到售后的日常运营场景。

**一句话定位**：Shopify 商家的 AI 运营助手 —— 装上就能用，越用越聪明。

**产品形态**：Shopify App（主入口）+ Web Dashboard（辅助）

### 2.2 工具详解

#### ① 商品优化器（Shopify 原生）

| 维度 | 说明 |
|------|------|
| 解决问题 | "我的商品标题/描述不够好，转化率低" |
| 数据来源 | **Shopify Product API**（直接读取店铺商品数据） |
| 核心能力 | AI 分析商品标题/描述/图片质量 → 自动优化建议 → 一键回写 Shopify |
| 优化维度 | SEO 标题优化、描述补充、标签建议、图片 Alt 文案 |

#### ② 品类调研器

| 维度 | 说明 |
|------|------|
| 解决问题 | "这个品类值不值得做？" |
| 数据来源 | Google Trends + Shopify 行业数据 |
| 核心能力 | 搜索趋势、竞争度、季节性分析 |
| 输出 | 品类调研报告（含趋势图、关键词列表、竞争评分） |

#### ③ 广告生成器

| 维度 | 说明 |
|------|------|
| 解决问题 | "我不会写广告文案，不懂怎么投放" |
| 数据来源 | **Shopify 商品数据** → 自动提取卖点 → 生成广告 |
| 核心能力 | 选择 Shopify 商品 → AI 生成广告标题+描述+关键词 → 一键创建广告 |
| 支持渠道 | Google Ads（搜索/购物/PMax）、Meta Ads、TikTok Ads |
| 对接方式 | 从 Shopify 商品出发，向多渠道输出 |

#### ④ 申诉信生成器

| 维度 | 说明 |
|------|------|
| 解决问题 | "账号/店铺被封了，不知道怎么申诉" |
| 模板类型 | Shopify 店铺封禁、GMC 封号、Google Ads 拒登、Meta 广告被拒、支付网关冻结等 |
| 核心能力 | 选择封号类型 → 输入店铺信息 → AI 基于成功案例生成申诉信 |
| 差异化 | **基于真实 Shopify 商家成功案例打磨的模板库** |

#### ⑤ Feed 优化器

| 维度 | 说明 |
|------|------|
| 解决问题 | "我的 Google Shopping / Meta 目录广告效果差" |
| 数据来源 | **Shopify Product Feed**（直接从店铺商品生成优化后的 Feed） |
| 核心能力 | 读取 Shopify 商品 → AI 优化标题/描述/分类/属性 → 生成优化 Feed → 同步到 GMC / Meta Catalog |
| 优化维度 | 标题关键词注入、描述补充、Google 分类映射、属性完善 |

#### ⑥ AI 博客生成器

| 维度 | 说明 |
|------|------|
| 解决问题 | "独立站需要内容但没人写" |
| 数据来源 | **Shopify 商品数据**（自动提取产品信息） |
| 核心能力 | 选择店铺商品 → AI 生成 SEO 友好的营销博客 → **一键发布到 Shopify Blog** |
| 内容类型 | 产品评测、使用指南、对比文章、行业趋势 |

#### ⑦ 促销活动生成器

| 维度 | 说明 |
|------|------|
| 解决问题 | "大促不知道怎么策划" |
| 数据来源 | **Shopify 历史销售数据** + 竞品分析 |
| 核心能力 | 输入活动类型 → AI 结合店铺数据生成促销方案 → 自动创建 Shopify 折扣码 |
| 输出 | 促销策略 + 折扣码 + 中英文文案 + Banner 建议 |
| 活动类型 | 黑五、圣诞、清仓、新品上市等 |

#### ⑧ 智能受众

| 维度 | 说明 |
|------|------|
| 解决问题 | "广告投给谁效果好？" |
| 数据来源 | **Shopify 客户数据 + 订单数据** → 分析购买画像 |
| 核心能力 | 基于店铺真实客户画像 → AI 生成受众推荐 → 可同步到 Google/Meta 广告平台 |
| 差异化 | 从 Shopify 真实交易数据出发，而非纯猜测 |

#### ⑨ 店铺周报

| 维度 | 说明 |
|------|------|
| 解决问题 | "每周看数据太累了" |
| 数据来源 | **Shopify Analytics API** + GA4（可选） |
| 核心能力 | 自动汇总店铺周数据 → AI 生成分析洞察 → 推送报告 |
| 输出 | 销售额、订单量、转化率、流量来源 + AI 异常分析 + 改进建议 |

#### ⑩ 社媒内容投放

| 维度 | 说明 |
|------|------|
| 解决问题 | "社交媒体要发内容但没精力" |
| 数据来源 | **Shopify 商品图片和信息** |
| 核心能力 | 选择商品 → AI 生成多平台营销内容（图文+短文案）→ 自动发布 |
| 支持平台 | Facebook、Instagram、TikTok、Pinterest |

## 三、上线优先级

| 批次 | 工具 | 核心理由 |
|------|------|---------|
| **第一批** | ③ 广告生成器 | 最高频刚需，Shopify 商品→广告一键生成 |
| **第一批** | ⑤ Feed 优化器 | Shopify Feed→GMC/Meta，差异化强 |
| **第一批** | ① 商品优化器 | 纯 Shopify 原生，安装即用，最快落地 |
| **第二批** | ⑥ 博客生成器 | 商品→博客→发布 Shopify，全在生态内 |
| **第二批** | ④ 申诉信生成器 | 痛点极强，Shopify 封店也纳入 |
| **第二批** | ⑨ 店铺周报 | Shopify 数据自动汇总，用户粘性工具 |
| **第三批** | ⑧ 智能受众 | 需要积累足够订单数据 |
| **第三批** | ② 品类调研器 | 补充选品决策 |
| **第三批** | ⑦ 促销活动生成器 | 配合大促节点上线 |
| **第三批** | ⑩ 社媒内容投放 | 多平台对接需要时间 |

## 四、商业模式

- **Shopify App Store 上架**：Shopify 商家直接安装
- **免费增值（Freemium）**：基础功能免费，高级功能按月订阅
- **团队/Agency 版**：多店铺管理、批量操作、白标

## 五、与方案二、三的关系

三个方案是**一个 Shopify App 的三层深度**：

```
第一层（工具）→ 方案一
  Shopify App 安装后即可使用的 AI 工具集
  商品优化、广告生成、Feed优化、博客、周报

第二层（广告闭环）→ 方案二（AdPilot）
  方案一的广告工具升级为完整投放平台
  Shopify 商品 → 生成 → 投放 → 追踪 → 优化

第三层（决策中台）→ 方案三（OpsGenie）
  Shopify 全链路数据 + AI 运营建议
  店铺健康分 → 每日建议 → 一键执行
```

### 功能归属对照

| 方案一的工具 | → 升级为方案二模块 | → 汇入方案三数据 |
|---------|-----------------|----------------|
| 广告生成器 | 素材工坊 + 智能投手 | 广告效果数据 |
| Feed 优化器 | 效果中心 | 产品表现数据 |
| 智能受众 | 智能投手（受众推荐） | 受众转化数据 |
| 店铺周报 | — | 健康体检数据源 |
| 品类调研 | — | 趋势预测数据源 |
