## 一、产品概述

**OpsGenie** 是一款面向跨境电商卖家的 AI 运营决策中台，接入 Shopify 店铺 + 广告平台 + GA4 数据，通过 AI 分析生成**每日可执行的运营决策建议**。

不只是仪表盘，而是一个**会思考的运营搭档**。

**一句话定位**：把 10 年运营老手的经验装进 AI，让每个卖家都有自己的「首席运营官」。

## 二、市场分析

### 2.1 为什么是「运营决策」？

当前 AI 在电商的应用集中在**营销端**（84% 品牌已使用），但在**运营决策端**仅有 33% 的渗透率：

| 场景 | AI 渗透率 | 工具成熟度 | 机会 |
|------|----------|-----------|------|
| 广告素材生成 | 高 | 成熟 | 🔴 红海 |
| 客服自动化 | 高 | 成熟 | 🔴 红海 |
| SEO 优化 | 中 | 较成熟 | 🟡 竞争中 |
| **库存/补货决策** | **低** | **不成熟** | 🟢 **蓝海** |
| **定价策略** | **低** | **不成熟** | 🟢 **蓝海** |
| **SKU 生命周期管理** | **极低** | **几乎空白** | 🟢 **蓝海** |
| **跨渠道运营优化** | **极低** | **几乎空白** | 🟢 **蓝海** |

### 2.2 目标用户

| 用户类型 | 画像 | 核心痛点 | 预期价值 |
|----------|------|----------|---------|
| **核心用户** | 年营收 $50 万-$5000 万的 DTC 品牌 | 数据散落各平台，靠经验决策，反应慢 | 数据驱动决策、提前预警 |
| **增值服务对象** | 代运营公司现有客户 | 每月只能看到运营报告，缺乏实时洞察 | 随时了解店铺健康度 |
| **扩展用户** | 代运营公司/TP | 管理多个客户店铺，需要统一视图 | 多店铺运营效率 |

### 2.3 产品差异化

**OpsGenie 的核心价值：**

1. **从「看数据」到「做决策」**：不只是 Dashboard，每天推送可执行的 Action Items
2. **跨平台全链路**：Shopify + Google + Meta + TikTok + GA4 数据融合分析
3. **行业模型**：基于 100+ 客户的运营数据训练，懂跨境电商的行业特性
4. **中国卖家场景**：理解中国供应链节奏、季节性、物流时效等特殊因素

**市场机会**：目前没有产品真正做到「数据进来 → AI 分析 → 告诉你该做什么 → 一键执行」的闭环。

### 3.2 功能详解

#### 模块一：健康体检

> 「你的店铺今天状态如何？」

每日自动生成的店铺健康报告：

- **综合健康分**（0-100）：基于 GMV、流量、转化率、退货率、广告 ROAS 等多维指标综合评分
- **流量诊断**：各渠道流量占比、环比变化、异常波动检测
- **转化漏斗**：访问 → 浏览 → 加购 → 下单 → 付款 → 收货 全链路转化率
- **财务概览**：收入、成本（广告费+产品成本+物流）、毛利、净利实时计算
- **客户健康**：新客/老客比例、复购率、客单价趋势、LTV 分布

**展现形式示例**：
```
📊 今日店铺健康分：78/100 (↑3)
━━━━━━━━━━━━━━━━━━━━━━━
✅ GMV $12,340 (↑8% vs 昨日)
⚠️ 转化率 2.1% (↓0.3%, 需关注)
✅ ROAS 3.2x (行业 avg 2.8x)
🔴 退货率 5.8% (超过 5% 阈值)
✅ 新客占比 62% (健康范围)
```

#### 模块二：AI 顾问

> 「今天你应该做什么？」

每日推送 3-5 条可执行建议，按优先级排序：

- **🔴 紧急**：退货率飙升 → 建议检查某 SKU 的产品描述/尺码表
- **🟡 重要**：某广告组 CTR 下降 30% → 建议更换素材或调整受众
- **🟢 机会**：发现某品类搜索量增长 50% → 建议增加该品类广告预算

**建议类型**：

| 类型 | 示例 | 数据来源 |
|------|------|---------|
| 定价建议 | "SKU-A123 在竞品降价后建议调至 $29.99" | Shopify + 竞对监控 |
| 库存预警 | "SKU-B456 按当前销速 12 天后断货" | Shopify 库存 + 销售预测 |
| 广告优化 | "Google 广告组 #3 的 ROAS 低于阈值，建议暂停" | Google Ads |
| 产品洞察 | "新品 C789 上架 7 天，收藏率远高于同类" | Shopify + GA4 |
| 客户运营 | "有 230 位客户 90 天未复购，建议触发召回邮件" | Shopify 客户数据 |
| 竞对动态 | "竞品 X 本周新增 3 个 SKU，且在 Google 投放增加" | 竞对监控 |

#### 模块三：预测引擎

> 「下周/下月会发生什么？」

- **需求预测**：基于历史销量、季节性、营销活动、外部趋势，预测 SKU 级别未来 30 天销量
- **库存预警**：结合需求预测 + 供应链时效，自动计算安全库存和补货时间点
- **趋势预判**：基于 Google Trends + 社媒热度 + 行业数据，识别即将爆发的品类/关键词
- **客户流失预警**：基于购买频次、浏览行为、客诉记录，预测即将流失的高价值客户
- **ROAS 预测**：基于当前投放趋势，预测月底的广告回报率

#### 模块四：执行中心

> 「一键把建议变成行动」

- **一键调价**：AI 建议调价后，一键同步到 Shopify
- **补货建议 → 采购单**：自动生成采购建议单（数量 + 时间 + 供应商）
- **广告预算调整**：直接调整 Google/Meta 广告预算
- **自动化规则**：设置触发条件，自动执行操作
  - "当某 SKU 库存 < 50 时自动提醒"
  - "当 ROAS < 2x 持续 3 天时自动暂停广告"
  - "当新品收藏率 > 10% 时自动增加广告预算"
- **邮件/通知触发**：基于 AI 建议自动触发客户邮件（Klaviyo 集成）

### 3.3 AI 对话能力

除了主动推送建议，用户还可以**直接问 AI**：

```
用户：我的黑五准备策略应该怎么做？

OpsGenie：基于你店铺过去两年的黑五数据分析：
1. 去年黑五 GMV $45K，同比增长 23%
2. 表现最好的品类是 X（占 40% 收入）
3. 建议提前 2 周开始预热广告（去年预热带来 35% 额外流量）
4. 库存建议：品类 X 备 2.5 倍日常库存，品类 Y 备 1.8 倍
5. 定价建议：基于竞品分析，折扣力度建议 20-25%
   （去年 30% 折扣导致毛利过低）
```

## 四、商业模式

- **SaaS 订阅制**：按店铺数量 + 功能层级阶梯收费
- **代运营增值模式**：作为代运营服务的附加产品，提升客户粘性和 ARPU
- **内部提效**：投手使用 OpsGenie 管理客户，运营效率提升

## 五、产品形态

- **Web SaaS**（主站）：独立 Web 平台，支持所有功能
- **Shopify App**（嵌入式）：在 Shopify 后台直接查看健康分和建议
- **移动端推送**（后期）：关键告警和每日建议推送到手机

## 六、与方案 A 的协同

如果 AdPilot（方案 A）和 OpsGenie 同时推进：

```
OpsGenie（决策中台）
    │
    ├── "你的 Google 广告 ROAS 正在下降"
    │     └── 一键跳转 → AdPilot 生成新素材并替换
    │
    ├── "品类 X 搜索趋势上升 50%"
    │     └── 一键跳转 → AdPilot 创建新广告组
    │
    └── "竞品正在加大 TikTok 投放"
          └── 一键跳转 → AdPilot 生成 TikTok 素材
```

**OpsGenie 是大脑，AdPilot 是手脚** —— 一个负责「想清楚」，一个负责「做到位」。

## 七、风险与应对

| 风险 | 概率 | 影响 | 应对 |
|------|------|------|------|
| AI 建议质量不靠谱 | 高 | 致命 | 初期以规则为主 + AI 润色；建立人工审核回路；收集反馈持续优化 |
| 数据接入复杂度高 | 中 | 高 | 优先 Shopify + Google，分阶段接入其他平台 |
| 市场需要教育 | 中 | 中 | 依托现有客户关系冷启动；用案例证明价值 |
| 产品太重、上手难 | 中 | 中 | MVP 聚焦「每日建议推送」单点突破；降低学习成本 |
| 与 ERP 产品定位混淆 | 低 | 中 | 明确定位：不做订单管理/仓储，只做「决策层」|

## 九、合作价值

### 对代运营公司的价值
- **从服务到产品**：新增 SaaS 订阅收入线，公司估值从「人头」模式升级为「产品」模式
- **客户粘性飞升**：数据 + 工具深度绑定，客户迁移成本极高
- **运营效率翻倍**：投手通过 AI 辅助，可管理 2-3 倍客户
- **品牌升级**：从「代运营服务商」升级为「AI 驱动的运营科技公司」
- **数据飞轮**：越多客户使用 → 越多数据 → 模型越准 → 产品越好 → 更多客户

### 长期想象空间
- 累积足够多的跨境电商运营数据后，OpsGenie 可成为行业级的「运营智能基础设施」
- 从独立站扩展到亚马逊、TikTok Shop 等全渠道
- 从运营决策扩展到供应链、财务、客户关系全域 AI
