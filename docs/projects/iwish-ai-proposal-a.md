## 一、产品概述

**AdPilot** 是一款面向跨境电商卖家的 AI 广告投放优化工具，帮助**自主投放广告**的中小 DTC 品牌实现：

- 🎨 AI 自动生成高转化广告素材（图片 + 文案 + 视频脚本）
- 🎯 智能出价与预算分配建议
- 📊 跨平台（Google / Meta / TikTok）效果归因与对比分析
- 🔄 自动化 A/B 测试与优化循环

**一句话定位**：让没有专业投手的卖家，也能投出专业水准的广告。

## 二、市场分析

### 2.1 目标用户

| 用户类型 | 画像 | 痛点 | 月预算 |
|----------|------|------|--------|
| **核心用户** | 自主投放的中小 DTC 品牌 | 不会优化素材、不懂出价策略、跨平台数据看不懂 | $2K-50K/月广告预算 |
| **次要用户** | 刚起步的独立站卖家 | 完全不懂广告投放，需要从零指导 | $500-2K/月 |
| **增长用户** | 代运营公司内部投手 | 需要提效工具，服务更多客户 | 多账户管理 |

### 2.2 市场规模

- 全球 Shopify 商家 400 万+，其中活跃投放广告的约 60-80 万
- 中国出海 DTC 品牌约 10 万+，自主投放占比约 40%
- AI 广告工具市场（$86.5B 大盘中的细分）年增长 35%+

### 2.3 竞品分析

| 竞品 | 定位 | 定价 | 优势 | 劣势 |
|------|------|------|------|------|
| **Pencil** (pencil.li) | AI 广告素材生成 | $119-999/月 | 素材生成质量好 | 仅素材，无投放优化；不针对中国卖家 |
| **AdCreative.ai** | AI 广告创意平台 | $29-149/月 | 价格低、模板多 | 缺乏投放数据闭环；泛用非电商垂直 |
| **Shopify Magic** | Shopify 原生 AI | 内置免费 | 零门槛 | 功能浅、不支持跨平台 |
| **Madgicx** | Meta 广告优化 | $44-299/月 | 自动化投放规则 | 仅 Meta；不生成素材 |
| **Triple Whale** | 电商归因分析 | $129-279/月 | 数据归因准确 | 仅分析不优化；不生成素材 |

### 2.4 差异化定位

**AdPilot 的差异点在于：**

1. **全链路闭环**：素材生成 → 投放建议 → 效果追踪 → 自动优化（竞品都只做其中一环）
2. **中国卖家本土化**：中文界面、支持人民币结算、理解中国供应链场景
3. **跨平台统一视图**：Google + Meta + TikTok 一站管理（多数竞品只做单一平台）
4. **艾维运营经验 AI 化**：基于 10 年真实投放数据训练的行业模型

### 3.2 功能详解

#### 模块一：AI 素材工坊

**核心功能**：
- **一键生成广告图**：输入产品链接/图片 → AI 自动生成多套广告素材（Banner、方图、竖图）
- **AI 文案生成**：根据产品特性 + 目标平台 + 目标市场，自动生成多语言广告文案
- **视频脚本生成**：基于产品卖点自动编排视频脚本（适配 TikTok / Reels / YouTube Shorts）
- **竞品素材库**：抓取同品类热门广告素材（合规展示），作为创意灵感参考
- **品牌风格记忆**：学习品牌调性（色彩、字体、语言风格），后续生成自动对齐

**技术方案**：
- 图片生成：Gemini / Stable Diffusion + ControlNet（保持产品主体一致性）
- 文案生成：GPT-4 / Claude + 行业 Prompt Engineering
- 多语言支持：集成翻译 API + 本地化润色模型

#### 模块二：智能投手

**核心功能**：
- **出价建议**：基于行业 benchmark + 历史数据，推荐 CPC/CPM 出价策略
- **预算分配**：多平台预算智能分配（Google 40% / Meta 35% / TikTok 25%）
- **受众推荐**：分析已有客户画像，推荐高转化受众定向
- **投放时段**：基于目标市场时区和历史转化数据，推荐最佳投放时段
- **一键创建广告**：在 AdPilot 内直接创建 Google/Meta/TikTok 广告（API 集成）

**技术方案**：
- Google Ads API + Meta Marketing API + TikTok Marketing API
- 基于历史投放数据的贝叶斯优化模型
- 行业基准数据库（按品类、地区、季节）

#### 模块三：效果中心

**核心功能**：
- **跨平台仪表盘**：Google / Meta / TikTok 数据统一展示
- **ROAS 实时追踪**：广告花费回报比实时计算
- **转化漏斗分析**：曝光 → 点击 → 加购 → 下单 全链路
- **归因分析**：多触点归因（首次点击、末次点击、线性归因、数据驱动归因）
- **竞品广告监控**：自动追踪竞品广告活动变化

**技术方案**：
- Shopify API（订单/转化数据）+ 各广告平台 Reporting API
- 自建归因模型（解决 iOS 14+ 后各平台归因不准的痛点）
- 数据仓库：ClickHouse（高性能分析查询）

#### 模块四：优化引擎

**核心功能**：
- **自动 A/B 测试**：AI 自动拆分流量，测试不同素材/文案/受众组合
- **素材疲劳检测**：自动识别 CTR 下降趋势，触发新素材生成
- **预算重分配**：表现好的广告组自动加预算，差的自动减
- **异常告警**：ROAS 骤降、花费异常、账户问题实时通知
- **周/月报告**：自动生成投放总结报告（可导出 PDF，发给客户/老板）

**技术方案**：
- 多臂老虎机算法（Thompson Sampling）
- 时序异常检测（Prophet / 自研模型）
- 自动化工作流引擎（Temporal / Bull Queue）

### 3.3 用户旅程

```
Day 1:  注册 → 绑定 Shopify 店铺 → 连接广告账户(Google/Meta/TikTok)
Day 1:  选择产品 → AI 生成第一批广告素材 → 审核确认
Day 2:  AI 推荐投放策略 → 一键创建广告
Day 3-7: 效果中心查看数据 → AI 给出优化建议
Day 7+: 优化引擎自动迭代素材和预算 → ROAS 持续提升
```

### 5.2 技术选型

| 层级 | 技术 | 理由 |

| 后端 | Ruby on Rails 7 | 团队熟悉（iwish-server 基础）、快速迭代 |

- ✅ 多平台广告数据聚合框架（Google/Meta/Bing/Criteo）

- ✅ 竞对监控（Playwright 截图）

| 周 | 任务 | 交付物 |
|----|------|--------|
| W1-2 | 项目搭建 + Shopify App 基础框架 | 能安装的 Shopify App |

- 视频素材 AI 生成
- 预算自动优化引擎
